Detailed Description

Container and creator for meshes.

A Mesh can consist of Vertices, Edges, Triangles, and nested Meshes. It provides functionality to conveniently create those objects.

In addition to creating the topological information of a mesh, the Mesh class can also be used to add data to the vertices of a mesh.

Usage example: precice::mesh::tests::MeshTest::testDemonstration()

void precice::mesh::Mesh::computeState ( )

Necessary before any geom. operations can be performed on the mesh.

If no edges (in 2d) or triangles(in 3d) are given, no normals are computed in order to avoid dividing by zero on normalization of the vertex normals.

Circumcircles of edges and triangles are computed.
